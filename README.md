## Selected work

### Boxhunt.pl

A multiplayer network I designed, developed, deployed, and operate independently.

- Built a custom modular Java server platform from scratch.
- Operates across a Velocity and Paper multi-server environment.
- Uses Redis and MongoDB for cross-server communication, caching, and persistence.
- Includes custom gameplay systems, configuration tooling, addon lifecycle management, and deployment workflows.
- Reached more than 1,300 concurrent players at peak.
- Maintained in production since 2022.

**Technologies:** Java, Paper, Velocity, Redis, MongoDB, Gradle, Docker, Linux

### TwemojiChat

Open-source client mod adding Twemoji autocomplete and rendering to Minecraft chat.

- Supports Fabric, Forge, and NeoForge across approximately 19 loader and version combinations.
- Includes a custom data pipeline for generating emoji font sheets and shortcode aliases.
- Uses GitHub Actions for builds and release verification.
- Published under the MIT licence.

[GitHub](https://github.com/Bruhdows/TwemojiChat) · [Modrinth](https://modrinth.com/mod/twemojichat)

**Technologies:** Java, Kotlin, Python, Gradle, GitHub Actions
